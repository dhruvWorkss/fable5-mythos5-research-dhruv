# Project Glasswing: Full Research

## What Is Project Glasswing?

Project Glasswing is a collaborative cybersecurity initiative launched by Anthropic on April 7, 2026. Its purpose: put the most capable AI model ever built to work defending critical software infrastructure before adversaries can weaponize equivalent technology offensively.

The name "Glasswing" refers to the glasswing butterfly — transparent wings that reveal internal structure. The metaphor: making software's hidden vulnerabilities visible.

## The Model: Claude Mythos Preview

The original Glasswing model was Claude Mythos Preview (`claude-mythos-preview`), an unreleased frontier model specifically deployed for this initiative.

### Demonstrated Capabilities

| Achievement | Significance |
|:------------|:-------------|
| Identified thousands of zero-day vulnerabilities across every major OS and browser | Scale no human team could achieve |
| Found a 27-year-old OpenBSD vulnerability | Survived millions of automated security tests undetected |
| Found a 16-year-old FFmpeg flaw | Present in virtually every media-processing pipeline |
| 83.1% on cybersecurity vulnerability reproduction benchmarks | Opus 4.6 scored 66.6% on the same test |
| Operates largely autonomously without human steering | Doesn't need a security researcher guiding each step |

### What This Means

Mythos Preview doesn't just find bugs — it finds bugs that decades of automated tools, manual audits, and bug bounty programs missed. The 27-year-old OpenBSD vuln is the clearest proof: OpenBSD is specifically engineered for security, has been continuously audited, and still harbored a flaw this model found.

## How Glasswing Relates to Mythos 5

Project Glasswing is the **access gateway** for Claude Mythos 5's cybersecurity capabilities:

```
Fable 5 (general public)
    → Cybersecurity classifier ON
    → Cannot perform offensive security tasks
    → Falls back to Opus 4.8 for security queries

Mythos 5 via Glasswing (approved partners)
    → Cybersecurity classifier OFF
    → Full offensive/defensive security capability
    → Same model weights as Fable 5
    → 30-day retention + logged access
```

## Why Glasswing Matters Strategically

### The Attacker-Defender Asymmetry

Historically, attackers have the advantage: they only need to find one vulnerability, while defenders must protect against all of them. Glasswing inverts this:

- The model finds vulnerabilities at a speed and scale that approaches comprehensive coverage
- Defenders get these findings first (restricted access)
- Attackers would need to build equivalent capability from scratch (Mythos-class models don't exist elsewhere publicly)

### The Clock Is Ticking

Anthropic's explicit framing: if they don't put this capability to defensive use first, adversarial nations and criminal groups will eventually develop equivalent models for offensive purposes. Glasswing is a preemptive defense program.

### Open Source Gets Protected

The Linux Foundation's inclusion (plus the $2.5M to Alpha-Omega/OpenSSF and $1.5M to Apache) means open-source software — which underpins most of the internet — gets AI-powered security hardening. This is the 16-year FFmpeg bug story: critical open-source infrastructure that nobody could afford to audit comprehensively now gets audited by an AI that doesn't get tired.
